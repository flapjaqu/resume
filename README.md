# Professional Resume Template

A modern, responsive resume template built with HTML and CSS. This template features a clean design with a gradient header, organized sections, and print-friendly styling.

## Features

- **Modern Design**: Clean, professional layout with gradient header
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Print-Optimized**: Special print styles for professional printing
- **Easy to Customize**: Simple HTML structure with clear placeholder content
- **All Sections Included**: Contact info, summary, experience, education, skills, projects, and certifications

## Customization Guide

### Quick Start

1. Open `index.html` in your preferred code editor
2. Replace all placeholder text with your personal information
3. Open `index.html` in a web browser to see your resume
4. Print or save as PDF using your browser's print function

### Section-by-Section Customization

#### Header & Contact Info
- Replace "Your Name" with your full name
- Update "Professional Title | Software Engineer" with your current role
- Update contact details (email, phone, location, LinkedIn, GitHub)

#### Professional Summary
Write a compelling 2-3 sentence summary highlighting:
- Years of experience
- Key achievements
- Career goals
- Core competencies

#### Professional Experience
For each position:
- Update job title, company name, and dates
- Replace bullet points with your actual achievements
- Use action verbs and quantify results when possible
- Focus on accomplishments, not just responsibilities

#### Technical Skills
Organize your skills into relevant categories:
- Programming Languages
- Frameworks & Libraries
- Tools & Technologies
- Databases
- Add or remove categories as needed

#### Education
Update with your:
- Degree and major
- University name
- Graduation date
- GPA (if above 3.0)

#### Key Projects
Showcase 2-3 significant projects:
- Project name and brief description
- Technologies used
- Your specific contributions
- Impact or results

#### Certifications & Awards
List relevant:
- Professional certifications
- Industry awards
- Academic honors
- Include issuing organization and year

## Styling Customization

### Colors
The main accent color is `#667eea` (purple-blue). To change the color scheme:
1. Open `style.css`
2. Find and replace `#667eea` with your preferred color
3. Update the gradient in the `.header` section if needed

### Fonts
The template uses Inter font from Google Fonts. To change:
1. Update the Google Fonts link in `index.html`
2. Modify the `font-family` property in `style.css`

## Printing Your Resume

1. Open `index.html` in Chrome, Firefox, or Safari
2. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (Mac)
3. Select "Save as PDF" as the destination
4. Adjust margins if needed (recommended: 0.5 inches)
5. Save your PDF resume

## Tips for a Great Resume

- **Keep it concise**: Aim for 1-2 pages maximum
- **Use action verbs**: Start bullet points with strong action words
- **Quantify achievements**: Use numbers and percentages when possible
- **Tailor content**: Customize for each job application
- **Proofread carefully**: Check for typos and grammatical errors
- **File naming**: Save as `FirstName-LastName-Resume.pdf`
